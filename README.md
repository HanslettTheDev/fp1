### Part 1: Microservices Design and Dockerization

### Installation

Ensure python is installed!. Then type the following in the command line

- For Windows

```cmd
python -m venv env
env\Scripts\activate
pip install -r requirements.txt
```

- For Linux

```bash
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
```
